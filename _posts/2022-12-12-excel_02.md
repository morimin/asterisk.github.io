<br><br>
데이터를 엑셀로 가져올때 '-' 가 뒤로 붙는 경우가 종종 있다. 이것을 다시 앞으로 오게 만드는 방법이다.
​<br><br>


-- 방법 :
```
= IF( ISNUMBER(셀번호1), 셀번호1, SUBSTITUTE( 셀번호2, "-", "") )
```
​<br>
​
-- 해석하면 :​<br>
``만약 셀번호1이 숫자이면, 셀번호1을 출력하고, 그렇지않으면 셀번호2를 출력하되 -를 제외하고 출력해라.``
​<br>
이다.

<br>
※ '-' ​를 앞에 붙이고 싶다면
```
= IF( ISNUMBER(셀번호1), 셀번호1, SUBSTITUTE( 셀번호2 , "-", "" ) * 1 )
```
을 해주면 된다.​
